# 📝 Feedback Form AI System 🚀

An AI-powered feedback automation system built using n8n that collects user feedback, intelligently classifies it using LLM (Groq AI), and automates responses and notifications via Slack, Gmail, and Airtable.

---

## ✨ Key Features

- 📋 Collects user feedback via form submission
- 🧠 Uses LLM (Groq AI) for intelligent classification:
  - Complaint
  - Compliment
  - Feature Request
- 📊 Stores structured feedback in Airtable
- 💬 Sends real-time notifications to Slack channels
- 📧 Sends automated email responses to users
- 🔄 Fully automated end-to-end workflow using n8n

---

## 🧠 Tech Stack

- n8n (Workflow Automation)
- Groq AI (LLM)
- Airtable (Database)
- Slack API (Notifications)
- Gmail API (Email Automation)
- LangChain AI Agent

---

## ⚙️ Workflow Overview

1. User submits feedback via form
2. n8n triggers the workflow automatically
3. AI Agent analyzes and classifies the feedback
4. Data is stored in Airtable for tracking
5. Slack notification is sent to team channels
6. Automated email response is sent to the user
